# 南京师范大学校园生活指南 AI Skill

![version](https://img.shields.io/badge/version-1.0.0-blue)
![license](https://img.shields.io/badge/license-MIT-green)

这是一个为南京师范大学（NJNU）师生和访客量身定制的 AI Skill。安装后，你的 AI 助手就能化身为“南师大百事通”，为你解答关于随园、仙林、紫金三个校区的各类校园生活问题。

## 🏫 关于南师大校区

本 Skill 覆盖南京师范大学的三个主要校区，并根据各校区特点提供精准信息：

| 校区名称 | 特点标签 | 适用场景 |
| :--- | :--- | :--- |
| **仙林校区** | 综合枢纽、生活运转、现代节奏 | 班车通勤、大型图书馆自习、食堂分布、社交活动 |
| **随园校区** | 人文底蕴、东方最美校园、慢生活 | 校园气质、老楼回廊、树影漫步、人文探访 |
| **紫金校区** | 任务导向、专业实践、连接现实 | 专业学习、实践操作、日常快节奏生活 |

## ✨ 这个 Skill 能做什么？

它能帮你快速获取准确的校园生活与服务信息：

- 🚌 **班车通勤查询**：查询三大校区间的班车线路、途经站点及节假日调整规则。
- 🍽️ **食堂美食指南**：了解各校区食堂分布、楼层特色档口、营业时间（含晚餐至 20:30）及高峰期避坑指南。
- 🛒 **生活配套定位**：快速找到 24 小时便利店（罗森、7-11等）、健身房、校医院门诊及自助洗衣房。
- 🆘 **应急求助指引**：提供失物招领途径、各校区保卫处 24 小时值班电话及医疗急诊联系方式。

## 💡 数据来源与可信度

本 Skill 的数据经过严格分层筛选，确保信息准确可靠：

- **官方权威**：学校历史、基础地址等采用官网口径。
- **实时服务**：班车、食堂、生活缴费等信息以官方服务平台 **“南师微服务”** 微信公众号为准。
- **经验参考**：包含部分学生视角的校园生活经验（如高峰期提示），让回答更接地气。

## 🚀 如何使用

在与 AI 对话时，你可以这样问：
- *“仙林校区有哪些食堂？可以用支付宝吗？”*
- *“我想从随园去仙林，班车怎么坐？哪里能查时刻表？”*
- *“半夜想买点东西，仙林校区有 24 小时的便利店吗？”*
- *“在校园里丢了东西该找谁？仙林校区保卫处电话多少？”*

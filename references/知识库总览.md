# 南师大知识库总览

这份参考不是百科堆砌，而是给写作提供稳定背景。

## 一、学校基础口径

- 学校名称：南京师范大学
- 常见简称：南师、南师大、NNU
- 学校历史源流公开常追溯到 1902 年创办的三江师范学堂
- 学校公开介绍中常强调其师范传统、人文底蕴和综合性发展

## 二、校区框架

本 skill 采用三校区框架：

- 随园校区
- 仙林校区
- 紫金校区

写作时不建议把三校区混写成一种气质，而应该根据任务选择主场景。

## 三、适合稳定调用的基础事实

- 南京师范大学当前公开资料中通常以三校区进行介绍
- 仙林校区常见公开地址为南京市栖霞区文苑路 1 号
- 随园校区常见公开地址为南京市鼓楼区宁海路 122 号
- 紫金校区常见公开地址为南京市玄武区板仓街 78 号

## 四、写作上的总分工

- 随园负责气质、人文、树影、老楼、回廊、慢下来
- 仙林负责通勤、课程节奏、图书馆、自习、社交、生活运转
- 紫金负责专业、实践、任务导向、连接现实场景

## 五、信息可信度分层

为避免把生活信息写错，本 skill 建议按下面三层取材：

### A 级：学校官网 / 学校官方公开页面

适合写：

- 学校历史
- 基础地址
- 官方名称
- 部门入口
- 学校级公开服务信息

### B 级：校内媒体或学校信息在外部平台的公开转载

适合写：

- 校园卡使用方式
- 暑期 / 假期服务安排
- 班车预约入口
- 校内报修入口
- 图书馆、门诊、值班信息

### C 级：公开生活信息与学生经验型内容

适合写：

- 食堂分布
- 宿舍感受
- 哪些场景更有学生日常感
- 校区生活气质

这类信息能用，但最好标明为“生活语境参考”，不要写成官方确定口径。

## 六、校园生活服务数据

### 班车信息

- **各线路起点与终点**：
  - 仙林校区 ↔ 随园校区
  - 仙林校区 ↔ 紫金校区
  - 随园校区 ↔ 紫金校区
- **途经站**：部分线路途经地铁站或南京南站等交通枢纽（具体按学期实时发布为准）。
- **节假日调整**：周末与节假日班次会大幅压缩，仅保留少量必要班次；寒暑假按专门的假期时刻表运行。
- **实时查询入口**：微信公众号“南师微服务” -> “校园服务” -> “校园班车”（高峰期需在线预约）。

数据来源：[南师微服务微信公众号](https://mp.weixin.qq.com/)

### 食堂信息

- **食堂数量**：仙林校区主要划分为东区、西区、南区、北区和新北区食堂；随园及紫金校区各设有集中食堂。
- **各食堂楼层分布**：通常分为多层，一层多为基础档口和大伙快餐，二层及以上常设有风味档口或教工餐厅。
- **营业时间**：早餐 06:30-09:00，午餐 10:30-13:00，晚餐 16:30-20:30。
- **特色档口**：苏式细点、南北面食、自选快餐等（具体网红档口随学期更迭）。
- **支付方式**：电子校园卡（南师微服务内绑定）、微信支付、支付宝支付。
- **高峰人流提示**：11:40-12:30 为人流最密集时段，建议错峰就餐。

数据来源：[南师微服务微信公众号](https://mp.weixin.qq.com/)
更新时间：2026-05-01

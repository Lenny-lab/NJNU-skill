# 三校区语境

这份参考服务于写作，不是官方导览词。

## 1. 随园

### 关键词

- 安静
- 老楼
- 树影
- 回廊
- 草地
- 坡道
- 人文气
- 历史感
- 慢下来

### 更适合的写法

随园不适合写得太热闹。它更像一个要慢慢走进去、慢慢看清楚的地方。写它时，可以先写第一眼的安静，再写停留之后才意识到的层次。

### 适合出现的动作

- 走进回廊
- 在树下停一会儿
- 傍晚从老楼边经过
- 在草地旁放慢脚步
- 在图书馆、教学楼和坡道之间来回经过

### 不建议

- 把随园写成旅游宣传片
- 只写“优雅”“古典”这种空词
- 忽略它作为当代校园仍在被使用的现实感

## 2. 仙林

### 关键词

- 通勤
- 日常运转
- 课程节奏
- 图书馆
- 食堂
- 校车
- 宿舍区
- 小组合作
- 群体性生活

### 更适合的写法

仙林更适合写“人怎么在这里一天一天地生活”。它的特点不主要靠情绪堆出来，而是靠上课、赶路、自习、集合、再出发这些动作构成。

### 适合出现的动作

- 赶去上课
- 和同学约自习
- 在图书馆和教学楼之间切换
- 计划一天行程
- 在不同任务之间调整节奏

### 不建议

- 把仙林写得太悬浮
- 全写成风景，而不写生活系统

## 3. 紫金

### 关键词

- 专业性
- 实践性
- 功能导向
- 任务明确
- 连接现实
- 行动感

### 更适合的写法

如果说随园偏气质、仙林偏日常，那么紫金更适合写成专业和实践导向更明显的空间。它不一定需要最强的抒情，而更适合写任务目标、学习应用和现实连接。

### 适合出现的动作

- 为某项任务集中准备
- 参加实践活动
- 面向现实问题展开行动
- 围绕专业目标进行安排

### 不建议

- 假装已经掌握大量细节而强行铺陈
- 把紫金写成随园或仙林的复制版

如果用户对紫金有个人经历，应优先采用用户自己的表述。

## 4. 跨校区写法建议

如果一段文字同时涉及三个校区，更自然的方式通常不是平均分配，而是：

- 随园负责记忆和气质
- 仙林负责日常与节奏
- 紫金负责专业与实践延伸

这样写更有层次，也更符合实际阅读感受。
